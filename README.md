# mbileGame
Описание проекта:
Это мобильная игра в жанре приключения с элементами экшена, где главный герой после кораблекрушения оказывается на таинственном острове и отправляется на поиски своей жены. По сюжету он исследует остров и пещеры, сражается с жуткими существами, находит оружие и постепенно раскрывает тайну острова. В конце игры герой сталкивается с главным боссом и получает выбор между хорошей и плохой концовкой. Сюжет строится вокруг выживания, исследования локаций, сражений и поиска выхода с острова.
